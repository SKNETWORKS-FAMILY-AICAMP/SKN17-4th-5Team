# 2. 프로젝트 개요
### 🔸프로젝트 명: 🚗 AI Driving Assistance 🚗 

### 🔸프로젝트 소개
**초보 운전자를 포함한 모든 운전자가 주행 중 겪는 불안과 혼란을 줄이고, 안전한 운전을 돕는 AI 기반 보이스봇·챗봇**
